### About 
lightweight White Knuckle plugin for smooth camera zoom. Hold a configurable key to zoom and **use the mouse wheel** to adjust zoom amount.

### Configuration Example
[General]

## Key used to activate zoom
# Setting type: String
# Default value: C
Zoom Key = G

## Maximum zoom amount
# Setting type: Single
# Default value: 100
Max Zoom = 300

## Speed of FOV change
# Setting type: Single
# Default value: 8
Zoom Speed = 4

## Smoothness factor for zoom interpolation
# Setting type: Single
# Default value: 8
Zoom Smoothness = 20

## Step size when adjusting zoom with mouse wheel
# Setting type: Single
# Default value: 5
Scroll Step = 1
